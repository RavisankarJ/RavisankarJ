Fishing the Numbers

Fish the numbers (using hook) as per the question displayed on top ride side.

Some questions are open ended. So there will be many answers for those questions. Player has to wait for the answer fish to appear and then catch it.

Question

  • The question will be displayed on the top right side.
  • An arrow will be shown each time the question changes.

How to fish the numbers?

  • Click / Touch on the fish which has the required number.
  • The hook will catch the number
  • Note: If you catch the wrong fish it will affect player's health.

Health, Level, Score and Attemps

  • You can view the player's health on top left corner. Player's health will decrease if you catch wrong fish.
  • You win if you give right answers all the attempts. The number of attempts will be displayed on top left side.
  • The current level will be displayed on top center. Each level will increase the difficulty.
  • The more you answer the more you collect shells. You can view your shells near the shell picture.
