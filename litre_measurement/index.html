Learn Litre Measurement
Let’s learn through play and let’s get assessed
How to use

This web application has 3 tabs. The student / teacher has to go to the puzzle tab and enter an ID. Then the user has to drag the correct jar to fill the tank according to the question displayed below the game window. Once answered all the questions teachers may download the result for analyzing purposes. Teachers can edit questions on the 'Edit Questions' tab.

Go to Puzzle
Audience

Mathematics teachers handling the preparatory level students (from 3rd Standard to 5th Standard).

Objective

Assess the students’ understanding of capacity units, unit onversion and four basic operations involving capacity. Assessment For, As and Of learning)

Software Requirements

Latest browser preferably chrome.

Features

  • The teachers may upload the questions using the ‘csv’ data format file.
  • The teachers, the head of the institution, the parents and the higher officials may get the students’ answers in ‘csv’ data format which can be used, to analyze where a student misunderstands the concept, to identify which students are behind the expected level etc. This feature makes this as an ‘assessment for learning’.
  • The questions may be typed in any language using Unicode.

Note: drag and drop the vessels to fill the tank. Fill the tank according to the question.
