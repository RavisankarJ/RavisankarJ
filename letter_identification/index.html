Letter Identification
Know your students' knowledge level in alphabets
How to use

This web application has 3 tabs. The student / teacher has to go to the play tab and enter an ID. Then the user has to listen the sound plays and select the correct alphabet. Once answered all the questions teachers may download the result for analyzing purposes. Teachers can edit words / letters on the 'Edit Questions' tab.

Play
Audience

English teachers handling the foundational and preparatory level students (from 1st Standard to 5th Standard).

Objective

Assess the students’ knowledge in alphabets. Assessment For, As and Of learning)

Software Requirements

Latest browser preferably chrome.

Features

  • The teachers may upload the questions using the ‘csv’ data format file.
  • The teachers, the head of the institution, the parents and the higher officials may get the students’ answers in ‘csv’ data format which can be used, to analyze where a student misunderstands the concept, to identify which students are behind the expected level etc. This feature makes this as an ‘assessment for learning’.
